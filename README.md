# image-processor

A library containing some image processing functionality created to test CI-CD pipelines

## Building

### Linux
```bash
make
```

### Windows (cross-compile)
```bash
make windows
```

### Both platforms
```bash
make both
```

### Debug build
```bash
make debug
```

### Configuration
```bash
make config
```
